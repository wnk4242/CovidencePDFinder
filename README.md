# Covidence PDF Finder Chrome Extension

This is a Chrome Extension allowing users to find PDF files for papers in the full-text screen stage.

## Install

1. Unzip this folder.
2. Go to `chrome://extensions/`.
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select this folder.

## Unpaywall email

Open `content.js` and replace:

```js
const UNPAYWALL_EMAIL = "YOUR_EMAIL@example.com";
```

with your real email address.
