# Privacy Policy for Covidence PDF Finder

Last updated: May 26, 2026

Covidence PDF Finder is a Chrome extension designed to help users locate and download article PDFs during full-text screening in Covidence.

This extension is an independent tool for Covidence users and is not affiliated with, endorsed by, or sponsored by Covidence.

## Information the extension uses

Covidence PDF Finder may read article metadata that is visible on Covidence study cards, including information such as article title, DOI, study number, author information, and publication year.

This information is used only to help users search for article PDFs, open fallback search pages, and create cleaner downloaded filenames.

## Optional Unpaywall email

Users may optionally save their own email address to enable Unpaywall API searches. This email address is stored locally in the user’s browser using Chrome storage and is sent only to Unpaywall when the user uses the PDF search feature.

Providing an Unpaywall email is optional. The extension can still be used without it.

## Local storage

The extension may store user preferences locally in the browser, including custom database search settings, hidden search options, and the optional Unpaywall email setting.

This information is stored locally through Chrome storage and is not sold or used for advertising.

## Third-party services

The extension may open or query third-party services to help users locate PDFs or article records. These services may include open-access sources, DOI pages, Google Scholar, Google Search, OpenAlex, ResearchGate, Semantic Scholar, institutional library databases, and Unpaywall.

When users choose to use these services, the relevant article title, DOI, or search query may be sent to the selected service as part of the search process.

## Data sharing

Covidence PDF Finder does not sell user data.

The extension does not use user data for advertising.

The extension does not transfer user data to third parties except when necessary to provide the user-facing PDF search and download features, such as sending a DOI or title to a selected search service.

## Limited Use disclosure

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
